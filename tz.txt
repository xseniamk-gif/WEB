1 Наименование проекта
Веб-приложение для продажи туров (автобусные туры, экскурсии, путешествия)

Цель проекта
Разработка полнофункционального веб-сайта на Python для онлайн-продажи туристических
путевок с возможностью поиска, фильтрации, бронирования и оплаты туров.

2.1 Роли пользователей
2.1.1 Гость (неавторизованный пользователь)
    Просмотр каталога туров
    Поиск и фильтрация туров
    Просмотр детальной информации о туре
    Просмотр отзывов
    Регистрация / авторизация

2.1.2 Авторизованный пользователь
Все возможности гостя

    Бронирование туров
    Просмотр истории бронирований
    Отмена бронирования
    Добавление отзывов и оценок
    Сохранение туров в избранное
    Управление личным профилем

2.1.3 Менеджер (администратор)
    Управление турами (CRUD операции)
    Управление бронированиями
    Подтверждение/отмена бронирований
    Управление отзывами
    Просмотр статистики продаж
    Управление пользователями

2.1.4 Суперадминистратор
Все возможности менеджера

    Управление ролями пользователей
    Настройка системы
    Просмотр системных логов

2.1.5 Поиск и фильтрация
    Параметры поиска:
    Поиск по названию и описанию

    Фильтр по типу тура
    Фильтр по стране
    Фильтр по городу отправления
    Фильтр по цене (диапазон)
    Фильтр по длительности
    Фильтр по дате отправления
    Сортировка: по цене, по дате, по рейтингу, по популярности

2.4 Личный кабинет пользователя
Разделы:
    Мои бронирования — список всех заказов с деталями
    Избранное — сохраненные туры
    Мои отзывы — написанные отзывы
    Профиль — личные данные:
    ФИО
    Email
    Телефон
    Дата рождения
    Паспортные данные (опционально)

2.5 Административная панель
Функции:
    Dashboard — графики продаж, популярные туры, статистика
    Управление турами — создание, редактирование, удаление, копирование
    Управление бронированиями — просмотр, изменение статуса
    Управление пользователями — блокировка, смена роли
    Управление отзывами — модерация
    Экспорт данных — выгрузка в Excel/CSV
    Управление контентом — главная страница, баннеры, акции
